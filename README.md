### nodejs数据抓取程序说明

本程序尝试抓取了http://www.win4000.com/mt/dilireba.html等几个网站的美图，你懂的。

#### 运行说明

后台命令行，直接node命令运行即可。

```bash
node app
```

#### 后续

后续会继续完善代码，添加前台代码，以开启服务端然后返回网页到前端的形式打开，直接输入服务器网址，然后在网页中输入想要抓取数据的网站，点击确定即可。
